# jQuery CDN Library

A collection of jQuery library versions hosted for easy CDN access via jsDelivr.

## üìö Available Versions

This repository contains 26 jQuery versions across three major releases:

- **v1.x** (11 versions) - Supports IE 6-8
- **v2.x** (3 versions) - No IE 6-8 support, smaller footprint
- **v3.x** (12 versions) - Modern, improved performance

## üìñ Usage Example

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>jQuery Example</title>
  </head>
  <body>
    <h1>Hello jQuery!</h1>
    <button id="myButton">Click Me</button>

    <!-- Include jQuery -->
    <script src="https://cdn.jsdelivr.net/gh/ramyibrahim-eg/jquery@main/v3/jquery-3.7.1.min.js"></script>

    <!-- Your custom script -->
    <script>
      $(document).ready(function () {
        $("#myButton").click(function () {
          alert("Button clicked!");
        });
      });
    </script>
  </body>
</html>
```

## ‚ö° Performance Tips

1. **Use the latest stable version** (3.7.1) for best performance and security
2. **Place script tags** before closing `</body>` tag for faster page loads
3. **Consider using** the `defer` or `async` attribute for non-critical scripts
4. **Cache effectively** - jsDelivr provides automatic caching and CDN benefits
